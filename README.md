# Smart-campus-event-management-System

A Smart Campus Event Management System designed to simplify event planning, registration, scheduling, and notifications within a campus environment. This project helps students and organizers manage events efficiently through a user-friendly platform.

## Features
- Event creation and management
- Student registration for events
- Event scheduling and notifications
- User-friendly interface
- Real-time event updates
